    Aula 10 - Como um site chega até você?

    Descubra

    Faça um teste.
    Abra seu navegador e agora digite qualquer endereço.
    Pode ser:
    https://google.com

    ou

    https://youtube.com

    ou

    https://smarteletrovini.com.br

    A página aparece quase instantaneamente.

    Mas existe uma pergunta muito interessante.

    Como seu computador encontrou exatamente esse site entre bilhões de páginas espalhadas pelo mundo?

    Hoje vamos acompanhar essa viagem.

    Imagine que você quer visitar um amigo

    Você sabe apenas o nome dele.
    Mas não sabe onde ele mora.
    Primeiro, você procura o endereço e depois, você segue até a casa. Com os sites acontece exatamente a mesma coisa.

    O navegador

    O navegador é o programa utilizado para acessar páginas da Internet.
    Exemplos:

    • Google Chrome;
    • Microsoft Edge;
    • Mozilla Firefox;
    • Safari.

    Ele funciona como uma porta de entrada para a Web. Quando você digita um endereço, o navegador inicia toda a comunicação.

    O que é um Domínio?

    Pense no domínio como o nome de uma empresa.
    Por exemplo https://smarteletrovini.com.br.

    Esse nome é fácil de lembrar, mas os computadores não trabalham com nomes, eles trabalham com números.

    O que é um endereço IP?

    Todo equipamento conectado à Internet possui um endereço, esse endereço chama-se IP (Internet Protocol).
    Podemos compará-lo ao endereço de uma casa. Assim como o carteiro precisa do endereço correto para entrega uma encomenda, os computadores precisam do endereço IP para localizar outro equipamento na rede.

    Quem descobre esse endereço?

    Entra em ação um servidor chamado DNS (Domain Name System). O DNS funciona como uma agenda telefônica da Internet.
    Você informa o nome, ele informa o endereço.
    Você digita: (www.smarteletrovini.com.br)[https://smarteletrovini.com.br]. O DNS localiza o endereço IP correspondente.

    Agora seu computador sabe exatamente para onde enviar o pedido.

    O servidor

    Depois de descobrir o endereço, seu navegador envia uma solicitação ao servidor.
    O servidor é um computador preparado para armazenar sites e responder às solicitações dos usuários.
    Imagine uma grande biblioteca. Você faz um pedido, o bibliotecário procura o livro e depois entrega exatamente o que você pediu. O servidor faz algo parecido.

    E onde estão as informações?

    Muitos sites utilizam um banco de dados. É nele que ficam armazenadas informações como:

    • Produtos;
    • Clientes;
    • Pedidos;
    • Notícias;
    • Comentários;
    • Usuários.

    Quando você pesquisa um produto em uma loja virtual, o servidor consulta o banco de dados para encontrar as informações solicitadas.

    Curiosidade

    Você pode acessar o mesmo site de qualquer lugar do mundo. Isso acontece porque o servidor permanece conectado à Internet aguardando solicitações dos usuários.
