# Aula 11 - Como os arquivos são organizados?
Imagine a seguinte situação. Você possui uma biblioteca com 10.000 livros, mas existe um problema.  
Todos os livros estão espalhados pelo chão, sem prateleira, sem etiquetas, sem ordem.  

Agora imagine que alguém peça: **“Encontre o livro de Matemática.”**
Quanto tempo levaria?  
Talvez horas, talvez dias.   
Agora imagine a mesma biblioteca, com tudo organizado, prateleira, categorias, etiquetas. Você encontra qualquer livro em poucos minutos.  
Seu computador funciona exatamente assim.

## O que é um arquivo?
Um arquivo é um conjunto de informações armazenadas.  
Pode ser:
- Uma fotografia; 
- Um vídeo;
- Uma música;
- Um documento;
- Uma planilha;
- Um programa.

Sempre que você salvar alguma informação, você cria um arquivo.  
 
## O que é uma pasta?
Imagine um armário, dentro dele existem várias gavetas. Cada gaveta possui documentos diferentes. As pastas funcionam da mesma maneira, elas servem para organizar arquivos.

### Exemplo 
Imagina seu computador, você possui:

- Fotos;
- Trabalho;
- Faculdade;
- Música.

Dentro de cada pasta existem arquivos relacionados ao assunto. Essa organização, facilita encontrar informações rapidamente.

## O nome também é importante
Imagine que você possua 500 arquivos, todos possuem o mesmo nome: "Novo documento".
**Como encontrar o correto?**
Por isso, escolher nomes claros faz parte da organização digital.

### Exemplo.
**Ruim**:

- Documento 1;
- Documento 2;
- Documento 3.

**Melhor**:

- Contrato cliente João.pdf;
- Relatório financeira Abril.xlsx;
- Apresentação Projeto.pptx;

## O que é a extensão do arquivo?
Todo arquivo possui um tipo.  
Esse tipo normalmente aparece no final do nome.

Exemplos:
- relatorios.pdf;
- foto.jpg;
- planilha.xlsx;
- musica.mp3;
- video.mp4.
Essas letras informam ao sistema operacional qual programa pode abrir aquele arquivo. 

## Como o sistema encontra seus arquivos?
Quando você salva um documento, o sistema operacional registra:

- Nome;
- Localização;
- Tipo;
- Tamanho; 
- Data.

Assim ele consegue localizar rapidamente o arquivo quando você precisar dele.

## O que acontece quando você exclui um arquivo?
Na maioria dos sistemas, o arquivo vai primeiro para a Lixeira.  
Isso permite recuperar informações apagadas por engano. Somente quando a lixeira é esvaziada ou o espaço é reutilizado é que a recuperação pode se tornar mais difícil.  

## Curiosidade
Muitas empresas perdem informações importantes, não porque o computador apresentou defeito. Mas porque os arquivos foram mal organizados. A organização também faz parte da segurança da informação.
