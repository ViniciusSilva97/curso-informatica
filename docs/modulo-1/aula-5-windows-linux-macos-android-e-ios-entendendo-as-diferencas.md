# Aula 5 - Windows, Linux, MacOS, Android e IOS: Entendendo as diferenças.

Imagine que você entrou em uma concessionária.  
Lá existem vários tipos de veículos.  

- Um carro popular;
- Um carro esportivo;
- Uma caminhonete;
- Um caminhão;
- Uma motocicleta.

## Qual deles é melhor?  

A resposta é: depende da necessidade. 
Um caminhão é excelente para transportar toneladas de carga.  
Mas seria péssimo para estacionar em um shopping.  
Um carro esportivo é rápido.  
Mas não é a melhor opção para uma fazenda.  
Com os sistemas operacionais acontece exatamente a mesma coisa.  
Não existe o melhor sistema operacional.  
Existe o mais adequado para cada situação.  

## O que é um sistema operacional? 
Na aula anterior aprendemos que o sistema operacional é o maestro do computador.  
Agora vamos conhecer os principais “maestros” do mundo da tecnologia.  

### Windows
O Windows é o sistema operacional mais utilizado em computadores pessoais.  
Por que ele é tão popular?  

Porque foi desenvolvido para facilitar a vida do usuário.  
A maioria dos programas comerciais funciona nele.  
Por isso encontramos Windows em:  

- Casas;
- Escolas;
- Empresas;
- Escritórios.

#### Pontos fortes

- Fácil de aprender;
- Grande compatibilidade;
- Grande quantidade de programas;
- Muito utilizado no mercado de trabalho.

#### Pontos de atenção
- Licença paga;
- Maior alvo de vírus devido à sua popularidade;
- Pode exigir hardware mais potente em versões recentes.

### Linux
Agora imagine um sistema que pode ser totalmente personalizado.  
Esse é o Linux.  
Ele é utilizado em:

- Servidores;
- Data centers;
- Supercomputadores; 
- Computação em nuvem;
- Dispositivos embarcados; 
- Empresas.

Você pode até não perceber, mas grande parte dos sites que você acessa diariamente roda em servidores Linux.  

#### Pontos forte 

- Gratuito;
- Seguro;
- Excelente desempenho;
- Muito utilizado por profissionais de TI.

#### Pontos de atenção

- Algumas distribuições exigem maior aprendizado;
- Nem todos os programas comerciais possuem versão nativa.

### macOS
O macOS é o sistema operacional dos computadores da Apple.  
Seu foco principal é oferecer uma experiência integrada entre hardware e software.  
É muito utilizado por profissionais de: 

- Design;
- Produção de vídeo; 
- Música;
- Desenvolvimento para dispositivos da Apple.

#### Pontos Fortes

- Excelente desempenho;
- Grande estabilidade;
- Boa integração com outros dispositivos da Apple.

#### Pontos de atenção

- Funciona oficialmente apenas em computadores Apple;
- Equipamentos costuma ter custo elevado.

### Android
O Android é o sistema operacional mais utilizado em smartphones. 
Ele é baseado no Linux.  
Está presente em aparelhos de diversos fabricantes.  

#### Onde encontramos Android ?

- Smartphones;
- Tablets;
- Smart TVs;
- Relógios inteligentes;
- Sistemas automotivos.

### iOS
O IOS é utilizado nos smartphones da Apple.  
Seu principal diferencial é a integração entre hardware e software, além do forte foco em segurança e experiência do usuário. 

## Existe um sistema melhor?
Não.  
Existe o mais adequado para cada necessidade.  

### Para jogar?  
O Windows costuma oferecer a maior compatibilidade com jogos.

### Para servidores
Linux é amplamente utilizado pela sua estabilidade e flexibilidade.

### Para uso doméstico
O Windows atende muito bem à maioria dos usuários.

### Para desenvolvedores de aplicativos Apple
macOS é o ambiente oficial.  

### Para smartphones
Android e iOS dominam esse mercado.

## Curiosidade 
Você provavelmente utiliza mais de um sistema operacional todos os dias.

### Por exemplo

- Notebook com Windows;
- Celular com Android;
- Caixa eletrônico com Linux;
- SmarTV baseada no Android;
- Servidores Linux quando acessa um site.

Mesmo sem perceber, você interage com diferentes sistemas ao longo do dia.
