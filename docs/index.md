# Curso de informática

Bem-vindo ao nosso curso de informática.

Este projeto foi criado para ensinar informática de maneira prática, técnica e acessível, conectando os conceitos teóricos ao funcionamento real dos computadores e das redes.

## O que estudaremos

Durante o curso, aprenderemos sobre:

- fundamentos da informática;
- componentes internos do computador;
- periféricos;
- sistemas operacionais;
- redes de computadores;
- segurança da informação;
- infraestrutura de tecnologia;
- diagnóstico e resolução de problemas.

## Nossas aulas combinarão

- explicações técnicas;
- exemplos do cotidiano;
- situações profissionais;
- atividades práticas;
- seções especiais **Dentro da caixa**.
